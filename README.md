# ねこビン

愛猫の毛を詰めて飾れる、かわいい猫型のメモリアルアイテムのランディングページです。

## 概要

このウェブサイトは、ねこビンの商品紹介ページです。チャームタイプとディスプレイタイプの2種類の商品と、ギフトセットを紹介しています。

## 機能

- レスポンシブデザイン対応
- スムーズスクロールナビゲーション
- 商品一覧から購入ページへの直接リンク

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript
